===> python supports 2 types of functions . These are:
                 1. built-in functions
                 2. user-defined functions
                          a).  normal functions
                          b).  recursive functions
                          c).  lambda functions


            

1. Built-in functions:
----------------------
===> The functions which are coming from along with python software.
     These functions are called built-in functions.
     
         EX:
             print(),input(),type(),id(),range(),
             len(),sum(),min(),max(),
             int(),float(),str(),complex(),bool(),
             list(),tuple(),set(),dict(),
             append(),insert(),extend(),pop(),remove(),clear(),
             copy(),reverse(),sort(),count(),index(),
             add(),update(),discard(),remove(),
             setdefault(),pop(),popitem(),keys(),values(),items(),get(),fromkeys(),
             lower(),upper(),title(),capitalize(),swapcase(),
             islower(),isupper(),istitle(),
             isdigit(),isspace(),isalpha(),isalnum(),
             split(),join(),replace(),
             lstrip(),rstrip(),strip(),format() ,......etc.
             



               
2. user-defined functions:
----------------------
===> The functions which are developed by programmer.
     These functions are called user-defined  functions.
 
===> syntax:
      def functionname(argumentslist):
          st-1
          st-2
          st-3
          st-4
          --
          --
          st-n
          return expression


 
    Here,
      ===> def means function definition (or) function declaration.
      ===> functionname reprsents name of the function.
      ===> argumentslist represents list of formal arguments
      ===> ':' indicates begining of the function
      ===> st-1,st-2,st-3,st-4,.... st-n are called body of the function
      ===> return expression,
           It is used to return values from function to main program.
           it is optional.
           It returns no value / one value/ morethan one value.

ex-3: write a   function  to find the square of a given number

    y = x*x
    return y

s = square(n)
print(f'Square value  = {s}')
