normal functions:
-------------------
1. All normal functions starts with def keyword.
2. functionname is compulsory.
3. argumentslist represents list of formal arguments
4. ':' indicates begining of the function.
5. st-1,st-2,st-3,...st-n are called body of the function.
6. return expression ,
   to return values from function to main program.
7. syntax:
    def functionname(argumentslist):
        st-1
        st-2
        st-3
        --
        --
        st-n
        return expression





lambda functions:
-------------------
1. All lambda functions starts with lambda keyword.
2. No need functionname.
   so lambda functins are also called nameless functions (or) anonymous functions.
3. argumentslist represents list of formal arguments
4. ':' indicates begining of the function.
5. only one statement is allowed.
6. return keyword is not allowed.
    
7. syntax:
         variablename = lambda argumentslist: st-1
         



ex-1: write a lambda function to find the square of a given number  
s = lambda x: x*x


n = 5
print('square value = ',s(n))
